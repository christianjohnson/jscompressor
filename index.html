An easy to use (Beta) JS Compressor.

This simple, easy to use (Beta) JS compressor is as easy as passing in a few parameters:

http://jscompressor.appspot.com/compress?code_url=http://rpidirectory.appspot.com/js/main.js
http://jscompressor.appspot.com/compress?js_code=alert%28%22hello%22%29%3B%2F%2F+This+comment+should+be+stripped
http://jscompressor.appspot.com/compress?js_code=var%20hi%20%3D%205%3B%0Aalert(hi)%3B&compilation_level=ADVANCED_OPTIMIZATIONS

Params:

  • code_url: url to the JS you want compressed. (optional)
  • js_code: actual JS code to be compressed (optional)
  • cache_time: time to cache code in seconds (optional; default 24 hours)
  • compilation_level: type of compression (optional; default: WHITESPACE_ONLY)
    • WHITESPACE_ONLY: Removes white space and comments.
    • SIMPLE_OPTIMIZATIONS: Changes variable names and function names.
    • ADVANCED_OPTIMIZATIONS: OMG awesome (use with caution - may break code).

Compression is done with Google's closure compiler. It caches your code for a default of 24 hours, so the subsequent requests will be extremely quick. Use it for development to minimize your JS, or directly include the link in a HTML file:

<HEAD>
<script src="http://jscompressor.appspot.com/compress?code_url=http://rpidirectory.appspot.com/js/main.js">
</HEAD>
